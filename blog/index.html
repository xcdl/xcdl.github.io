This site uses the GitHub Wiki-like theme by Liviu Ionescu.

License changed from LGPL to MIT License

Aug 4, 2016 - To further facilitate the integration of XCDL into other commercial or open source projects, the project license was changed from the copyleft LGPL to the permissive MIT License.

The GitHub Wiki content was migrated to GitHub Pages

Oct 12, 2015 - Reaching the limitations of GitHub Wikis, the documentation site was migrated to GitHub Pages. For a nice look, several plug-ins are required, so the site needs to be generated off-line.

The XCDL Wiki content was migrated from MediaWiki to GitHub Wiki

Sep 26, 2015 - Based on the general trend towards GitHub, the migration of the XCDL project to GitHub continued with the conversion of the MediaWiki pages to markdown, to be shown as GitHub Wikis.
